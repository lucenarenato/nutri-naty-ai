Desenvolvi um sistema completo de nutrição inteligente, que funciona de ponta a ponta. Criei um backend em Node.js com Express que roda uma API REST na porta 5000, seguindo arquitetura MVC com controllers, models, services e validação de dados. No frontend, construí uma aplicação React com Vite na porta 5173, usando Tailwind CSS para o design e Context API para gerenciar o estado da aplicação.

A parte principal foi integrar com a API da GROQ usando o modelo Llama 3.3-70B de IA. Implementei um sistema onde o usuário preenche seus dados pessoais - idade, peso, altura, nível de atividade física e objetivo - e o sistema processa essas informações, calcula o IMC automaticamente e envia para a IA gerar um plano alimentar personalizado. Trabalhei bastante no prompt engineering para garantir que a IA retornasse as informações num formato estruturado e útil.

O frontend tem um formulário responsivo que funciona bem em celular, tablet e computador, com validação em tempo real e feedback visual. Quando o usuário clica para gerar o plano, mostro estados de loading, e depois exibo o plano completo que pode ser copiado ou baixado como arquivo de texto. Configurei um proxy no Vite para facilitar a comunicação entre frontend e backend durante o desenvolvimento.

Também implementei tratamento de erros tanto no lado do cliente quanto do servidor, caso a API da GROQ falhe ou o usuário digite dados inválidos. A arquitetura ficou organizada com componentes reutilizáveis no React e uma separação clara de responsabilidades no Node.js. Todo o código está versionado no GitHub com um README detalhado explicando como configurar e rodar o projeto localmente.
